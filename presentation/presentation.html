To-do App Presentation

Project Description

This To-do App allows users to manage their tasks efficiently. It features user authentication, a clean interface, and supports both English and Arabic languages.

Screenshots

Features

  • User Registration and Login
  • Add, Edit, and Delete Tasks
  • Mark Tasks as Completed
  • Filter Tasks by Priority
  • Dark and Light Mode Support

Technologies and Techniques Used

This app was built using the following technologies:

  • React
  • TypeScript
  • Redux for state management
  • Material UI for styling
  • i18next for internationalization
  • React Hook Form for form handling
  • Redux Persist for state persistence
  • React Router DOM for navigation

Upcoming Features

In future updates, we plan to implement the following features:

  • Collaboration Tools for Users
  • Notifications for Task Reminders
  • Integration with Calendar Apps
  • Data Export Options
